# CTPortalEndUsers.ConsultantGoalsApi
CT(Candidates Tracking) Portal End Users - B2B End Users / Candidates / Applicants Consultant Goals API Service.

## Getting started

Clone an existing Git repository with the following command:

1. cd ../workspace/CTPortal/Candidates/Services/
2. Clone the repo using following git command: `git clone https://github.com/MetrixIT-Solutions/CTPortal.CandidatesSrvc.UserGoals.git`

3. Go to the appropriate directory to start the Api service: `cd CTPortal.CandidatesSrvc.UserGoals`
4. Node version `>=20.16.0` and npm version `>=10.8.1`
5. Run `yarn` or `npm install` to install dependencies
6. Run the command: `npm run start`
7. You can find the sevice status on browser with the url `http://localhost:3204`

Push an existing Git repository with the following command:

cd existing_repo
git remote add origin https://github.com/MetrixIT-Solutions/CTPortal.CandidatesSrvc.UserGoals.git
git branch -M main
git push -uf origin main
```
